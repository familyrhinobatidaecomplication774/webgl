## 🧪 If the screen stays blank

Try these steps:

1. Refresh the page or restart the app.
2. Close other tabs that use graphics.
3. Make sure hardware acceleration is on in your browser.
4. Update your graphics driver.
5. Try a different browser that supports WebGL.
6. Open a sample scene first, then load your own work.

Blank screens often come from browser or driver issues, not the app itself.
